# Geckos-Team-19

## Project setup
### Install dependencies
```
yarn
```
OR
```
npm install
```

### Database connection
* Create database.js config file
```
cp .env-sample .env
```

* Edit .env and change 'change_me' strings to  valid values

### Running the server
Starting the server
```
yarn server
```
OR
```
npm run server
```

Full project setup
```
yarn setup
```
OR
```
npm run setup
```
