# Geckos-Team-19

## Project setup
### Install dependencies
```
yarn
```
OR
```
npm install
```

###Setting up database
Create datbase.js config file
```
cp ./config/database-sample.js ./config/database.js
```

* Edit ./config/database.js and fill valid values

* Running the server
```
yarn server
```
OR
```
npm run server
```
